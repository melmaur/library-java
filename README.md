# 📚 Library Management System

A Java console application to manage a book library, built as part of a Java OOP learning portfolio.

## Features
- Add and remove books
- Search books by author
- Display all available books
- Persistent storage via CSV file

## Technologies
- Java 24
- IntelliJ IDEA
- Git & GitHub

## Agile Process
This project was developed following the **Scrum framework**:
- Work organized in a Sprint with a defined Sprint Goal
- User stories tracked in the Product Backlog
- Daily standup log maintained throughout development
- Sprint concluded with a Review and Retrospective

> Junior Developer holds a Professional Scrum Master I (PSM I) certification issued by Scrum.org in Fall 2025

## How to Run
1. Clone the repository
   git clone https://github.com/melmaur/library-java
2. Open the project in IntelliJ IDEA
3. Run Main.java
4. Use the console menu to interact with the library

## Project Structure
```
src/
└── library/
    ├── Book.java       → Book model with attributes and getters
    ├── Library.java    → Core logic — add, remove, search, CSV I/O
    └── Main.java       → Entry point with interactive console menu
```
